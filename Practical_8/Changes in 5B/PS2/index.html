QUESTION:

Write a code in any programming language that converts binary number to decimal number.

Solution:

          // C++ program to convert binary to decimal
          #include < bits/stdc++.h>
          using namespace std;

          // Function to convert binary to decimal
          int binaryToDecimal(int n)
          {
          	int num = n;
          	int dec_value = 0;

          	// Initializing base value to 1, i.e 2^0
          	int base = 1;

          	int temp = num;
          	while (temp) {
          		int last_digit = temp % 10;
          		temp = temp / 10;

          		dec_value += last_digit * base;

          		base = base * 2;
          	}

          	return dec_value;
          }

          // Driver program to test above function
          int main()
          {
          	int num = 10101001;

          	cout << binaryToDecimal(num) << endl;
          }
To compile use Ctrl + Enter on your device.

Sample Input:

Input : 111
Output : 7
Input : 1010
Output : 10
Input: 100001
Output: 33

num

temp

Complexity of the Code:

The complexity of this algo algorithm is O (Log2 n)
The worst time complexity of Quick Sort is O (n2).
